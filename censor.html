About Me

Roblox Rap Battles and I have a history going all the way back to January 12th 2019 so it wasn't too hard to write about. But I am not a part of the new generation of Roblox rappers, so a difficult part of this project was doing the research on how their lives are in todays world. I feel I really gained a special point of view on the topic thanks to making this website. If you're wanting to check out the game yourself you are able to do Here, if you end up playing the game it is recommended that you stick to the rules this tutorial has provided as they were given by master Roblox rappers. My name is Evelc Neerg and a struggle I had making this website was learning the java script needed to make my website look and act how I imagined it being.
